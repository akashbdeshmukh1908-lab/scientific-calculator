## Project Scope

The Scientific Calculator project is designed to perform both basic and advanced mathematical calculations. It provides a user-friendly graphical interface that allows users to perform arithmetic operations, trigonometric functions, logarithmic calculations, exponentiation, square root calculations, and other scientific computations. The project demonstrates the implementation of Java GUI components, event handling, and mathematical functions in a practical application.

## Features

* Addition, Subtraction, Multiplication, and Division
* Square Root Calculation
* Power and Exponential Functions
* Logarithmic Functions
* Trigonometric Functions (Sin, Cos, Tan)
* User-Friendly Graphical Interface
* Error Handling for Invalid Inputs
* Clear and Reset Functionality

## Technologies Used

* Java
* Swing (GUI)
* AWT
* Object-Oriented Programming Concepts

## How to Run

1. Install Java JDK.
2. Compile the program:

   ```
   javac ScientificCalculator.java
   ```
3. Run the program:

   ```
   java ScientificCalculator
   ```
4. The calculator window will open and be ready for use.

## Output

The application displays a scientific calculator interface where users can perform various mathematical operations and obtain results instantly.
